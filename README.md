# Orbit Mail

Orbit Mail is a responsive Flutter starter for a cross-platform email client.
It targets web, Windows, macOS, Android, and iOS from the same codebase.

## What is included

- A responsive inbox shell for desktop and mobile
- Folder navigation, message list, and message detail views
- A compose flow that saves a local draft placeholder
- Sample mail data that can later be replaced with Gmail, Microsoft 365, or IMAP integration

## Run it

```bash
flutter pub get
flutter run
```

## Build targets

- Web: `flutter run -d chrome`
- Windows: `flutter run -d windows`
- macOS: `flutter run -d macos`
- Android: `flutter run -d android`
- iOS: `flutter run -d ios`

## Next step
